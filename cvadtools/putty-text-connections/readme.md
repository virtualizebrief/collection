# Putty Text Connections

Place for simple menu or single launch of an ssh connection. Especially helpful for end users who have no idea what Putty, SSH, terminal are and taylored for Citrix application publishing.

## Two ways to do the same thing

- [Menu system](menu/) single pain of glass for all sessions
- [Single session](single/) direct launch for one connection
